## Proposal: Manufacturing Software Like Chips

**Track:** Platform Engineering  
**Format:** Lightning Talk, 5 minutes  
**Level:** Any

What if software is manufactured like semiconductors?

We still design software as if humans are the primary producers of code. When code becomes cheap, the bottleneck moves to design, verification, quality control, traceability, and manufacturing systems.

This talk maps concepts from industrial manufacturing to cloud-native technologies: GitOps, OCI artifacts, software supply chains, platform engineering, Kubernetes, Harbor, and agent runtimes. The goal is to turn code into a build artifact and platforms into software factories.

Core idea: AI agents do not need to understand the business by vibes. They need to understand the specification, constraints, and acceptance checks.
